# Project Miner - Idle Browser Game

## Table of Contents

## Introduction

**Project Miner** is a simple idle web-browser game ("idler"). Its main purpose is to setup a successful mining expedition and surpass your rivaling factions. You can fulfill this objective by following mechanics:
* Ship Management;
* Wise Fuel Handling;
* Establishing mines on planets in the Solar System;
* Succeeding in Extraction Missions;
* Managing your resources

## Technologies
The project is created with following technologies;
* React 19.2
* Tailwind CSS 4.1
* Express 5.2
And written using Typescript as a leading language.
